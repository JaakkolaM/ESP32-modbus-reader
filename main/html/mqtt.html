When MQTT is enabled and configured, this device will automatically publish Home Assistant discovery messages for all configured Modbus registers. Each register will appear as a sensor, switch (for writable coils), or number entity (for writable registers) in Home Assistant.

Note: Make sure your MQTT broker is configured in Home Assistant for auto-discovery to work.
